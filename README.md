广告转化预测 + LLM 助手化项目


项目简介

独立完成一个"广告转化预测 + LLM 助手化"的入门项目：
1. 用传统机器学习跑通 CVR 预测基本流程（数据读取→清洗→特征→训练→评估→解释）
2. 用阿里巴巴通义千问（Qwen）API 将流程产品化：自动生成 EDA 报告、自动产出特征说明、自动生成训练配置与实验记录、提供"业务同学可用的问答式分析助手"

将API Key配置到环境变量

1. 在Windows系统桌面中按Win+Q键，在搜索框中搜索编辑系统环境变量，单击打开系统属性界面。
2. 在系统属性窗口，单击环境变量，然后在系统变量区域下单击新建，变量名填入DASHSCOPE_API_KEY，变量值填入您的DashScope API Key。
3. 依次单击三个窗口的确定，关闭系统属性配置页面，完成环境变量配置。
4. 打开CMD（命令提示符）窗口或Windows PowerShell窗口，执行  echo %DASHSCOPE_API_KEY%  命令检查环境变量是否生效。
